## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript microservice sleeper project

## Stack

```bash
- NodeJs / pnpm
- Typescript
- PostgreSQL
- Pino Logger
- Docker
- Stripe
- Passport
```

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
